Totem Browser Extension

A complete cryptographic wallet extension for the Minima blockchain, implementing WOTS signatures (w=8 parameter, 90-element signatures) with hierarchical tree key management.

Overview
Test Extension

Key Features

  • WOTS Cryptography: Quantum-resistant signatures (w=8 parameter) with 3-level hierarchical trees
  • Secure Keystore: PBKDF2 + AES-GCM encryption with password protection
  • BIP39 Support: Standard mnemonic seed phrase generation and import
  • Transaction Approval: Complete transaction review with decoded inputs/outputs
  • Witness Building: Automatic cryptographic proof generation for blockchain submission
  • Network Integration: MEG Wallet and Axia API connectivity

Transaction Approval Flow

  • Decode transaction inputs and outputs with address validation
  • Display burn fees and network costs clearly
  • Show complete JSON witness data for transparency
  • Secure leaf reservation with mutex protection
  • Generate cryptographic proofs and submit to blockchain

Technical Implementation

  • TypeScript: Fully typed codebase with comprehensive error handling
  • React UI: Modern popup interface with dark theme
  • Webpack Build: Optimized extension bundling with multiple entry points
  • Chrome Extension API: Background scripts, content scripts, and popup integration
  • Provider API: Standard wallet methods for dApp integration

Installation Guide

To test the extension:
1. Open Chrome and go to chrome://extensions/
2. Enable "Developer mode" (toggle in top right)
3. Click "Load unpacked" and select the dist/ folder
4. The extension should appear in your browser toolbar
5. Return to this page and use the test buttons above
