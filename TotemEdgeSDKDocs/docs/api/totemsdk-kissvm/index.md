# @totemsdk/kissvm

KISSVM v1 evaluator for Minima scripting language — backed by Rust/WASM (v0.2+). The evaluator is available in two engines: Rust/WASM (default) and TypeScript (fallback).

- [buildWitness](functions/buildWitness.md)
- [evaluateScript](functions/evaluateScript.md) — TypeScript engine
- [evaluateScriptWasm](functions/evaluateScriptWasm.md) — Rust/WASM engine
- [parseScript](functions/parseScript.md) — TypeScript parser
- [parseScriptWasm](functions/parseScriptWasm.md) — Rust/WASM parser
- [sigdig](functions/sigdig.md)
- [simulateSpend](functions/simulateSpend.md)
